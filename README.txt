SITE CHÂTEAU DE POUGET — MODE D'EMPLOI

1. Ouvrez index.html dans votre navigateur pour voir le site.
2. Tous les fichiers doivent rester ensemble dans le même dossier.
3. Pour le publier gratuitement : déposez le dossier sur Netlify Drop, GitHub Pages ou Cloudflare Pages.
4. Le formulaire est actuellement une démonstration visuelle. Pour recevoir les demandes, connectez-le à Formspree, Netlify Forms ou à votre outil de réservation.
5. Modifiez les textes, coordonnées et mentions légales directement dans index.html.
